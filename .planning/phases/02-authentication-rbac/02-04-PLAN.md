<task type="auto">
  <name>Task 1: API client and auth utility functions</name>
  <files>
    frontend/src/lib/api.ts
    frontend/src/lib/auth.ts
  </files>
  <action>
    1. Create `frontend/src/lib/api.ts`:
       - Export `API_BASE_URL` = `process.env.NEXT_PUBLIC_API_URL || 'http://localhost:8000'`
       - Export `apiFetch` wrapper around fetch:
         - Prepends API_BASE_URL to relative paths
         - Always includes `credentials: 'include'` (sends HttpOnly cookies)
         - Always includes `Content-Type: application/json` for non-GET
         - Accepts typed generic for response: `apiFetch<T>(path, options): Promise<T>`
         - On 401 response: attempt silent refresh via `/api/auth/refresh/`, retry original request once
         - If refresh also fails: redirect to /login

    2. Create `frontend/src/lib/auth.ts`:
       - `login(username: string, password: string, rememberMe: boolean): Promise<LoginResponse>`:
         - POST to `/api/auth/login/` with {username, password, remember_me: rememberMe}
         - Returns {user: UserProfile} on success
         - Throws with error message on failure
       - `logout(): Promise<void>`:
         - POST to `/api/auth/logout/`
         - Redirect to /login via window.location (hard redirect to clear state)
       - `refreshToken(): Promise<boolean>`:
         - POST to `/api/auth/refresh/`
         - Returns true on success, false on failure
       - `getMe(): Promise<UserProfile>`:
         - GET to `/api/auth/me/`
         - Returns user profile with role
       - Type definitions:
         - `UserProfile = { id: number; username: string; email: string; first_name: string; last_name: string; role: { id: number; name: string }; is_force_password_change: boolean }`
         - `LoginResponse = { user: UserProfile }`
  </action>
  <verify>
    - `grep "credentials.*include" frontend/src/lib/api.ts` confirms cookie-based auth
    - `grep "api/auth/login" frontend/src/lib/auth.ts` confirms login function
    - `grep "api/auth/logout" frontend/src/lib/auth.ts` confirms logout function
    - `grep "api/auth/refresh" frontend/src/lib/auth.ts` confirms refresh function
    - `grep "UserProfile" frontend/src/lib/auth.ts` confirms type definition
  </verify>
  <done>API client sends credentials with every request. Auth utility provides login, logout, refresh, getMe functions. Auto-refresh on 401 with retry.</done>
