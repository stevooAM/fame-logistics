       - `AuthContext` with React.createContext:
         - `user: UserProfile | null`
         - `loading: boolean`
         - `logout: () => Promise<void>`
         - `refreshUser: () => Promise<void>`
       - `useAuth()` hook — `useContext(AuthContext)`, throws if used outside provider
       - `AuthProvider` component:
         - On mount: call `getMe()` to load user profile from /api/auth/me/
         - If getMe fails (401): redirect to /login (session expired or not authenticated)
         - While loading: show a full-screen loading spinner (centered, teal color)
         - If user has `is_force_password_change === true`: redirect to /change-password (create placeholder page or handle inline)
         - Wire `useIdleTimeout` hook:
           - onWarning: set `showSessionWarning = true`
           - onTimeout: call `logout()` and redirect to /login
           - onActivity: (optional) no-op, timer resets internally
         - Wire `setupTokenRefresh()` from api.ts on mount, cleanup on unmount
         - Render `SessionWarningDialog`:
           - open = showSessionWarning
           - onStayLoggedIn: call refreshToken(), resetTimers(), set showSessionWarning = false
           - onLogout: call logout()
         - Wrap children in `AuthContext.Provider` with user, loading, logout, refreshUser

    2. Create `frontend/src/middleware.ts` (Next.js edge middleware):
       - Match all routes EXCEPT: /login, /forgot-password, /reset-password, /_next, /favicon.ico, /api
       - Check for `access_token` cookie on the request
       - If cookie missing: redirect to /login
       - If cookie present: allow request through (actual token validation happens server-side on API calls)
       - Export `config = { matcher: ['/((?!login|forgot-password|reset-password|_next|favicon.ico|api).*)'] }`
       - Note: This is a lightweight check — the AuthProvider does the real validation via getMe()

    3. Update `frontend/src/app/(dashboard)/layout.tsx`:
       - Wrap children with `<AuthProvider>` inside the existing layout
       - Import AuthProvider from providers/auth-provider
       - Keep existing Sidebar + main structure
