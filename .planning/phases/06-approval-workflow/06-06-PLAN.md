    Implement the method:

    ```python
    def get_rejection_reason(self, obj) -> str | None:
        """
        Return the rejection_reason from the most recent REJECTED ApprovalQueue
        entry for this job, or None if the job has never been rejected.

        Uses the 'approval_requests' reverse relation (related_name on ApprovalQueue.job FK).
        Orders by -created_at so the most recent rejection is first.
        """
        latest = (
            obj.approval_requests
            .filter(status=ApprovalQueue.REJECTED)
            .order_by("-created_at")
            .values_list("rejection_reason", flat=True)
            .first()
        )
        return latest if latest else None
    ```

    Do NOT add `select_related` or `prefetch_related` to the method — the detail view fetches a single object, one extra query is acceptable.
