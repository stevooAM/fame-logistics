Add to core/views.py:

1. **ActiveSessionListView** (APIView, permission_classes=[IsAdmin]):
   - GET /api/sessions/ — List active sessions
   - Query OutstandingToken objects that are NOT blacklisted and not expired:
     ```python
     from rest_framework_simplejwt.token_blacklist.models import OutstandingToken, BlacklistedToken
     from django.utils import timezone

     active_tokens = OutstandingToken.objects.filter(
         expires_at__gt=timezone.now()
     ).exclude(
         id__in=BlacklistedToken.objects.values_list('token_id', flat=True)
     ).select_related('user').order_by('-created_at')
     ```
   - Group by user (one entry per user showing their most recent token)
   - For each active session, return: token_id (OutstandingToken.id), user_id, username, full_name (first + last), role (from UserProfile), created_at (login time), expires_at
   - Also try to find last LOGIN audit log entry for that user to get IP address
   - No pagination needed (active sessions should be small list <50)

2. **TerminateSessionView** (APIView, permission_classes=[IsAdmin]):
   - POST /api/sessions/{token_id}/terminate/ — Terminate a specific session
   - Find the OutstandingToken by id, blacklist it (create BlacklistedToken entry)
   - Also blacklist ALL outstanding tokens for that user (terminate all their sessions, not just one)
   - Log audit entry: action="DELETE", model_name="Session", object_repr="Terminated sessions for {username}"
   - Return 200 with {message: "Session terminated"}

Add to core/serializers.py:

3. **ActiveSessionSerializer**:
   - Fields: token_id, user_id, username, full_name, role (string), ip_address, created_at, expires_at

Add URL patterns to core/urls.py:
```
path("sessions/", ActiveSessionListView.as_view(), name="active_session_list"),
path("sessions/<int:token_id>/terminate/", TerminateSessionView.as_view(), name="terminate_session"),
```
