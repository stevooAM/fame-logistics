Add to core/views.py:

**AuditLogListView** (APIView, permission_classes=[IsAdmin]):
- GET /api/audit-log/ — returns paginated audit log entries (page_size=25)
- Filter query params:
  - ?user={username} — filter by acting user's username (case-insensitive contains)
  - ?action={CREATE|UPDATE|DELETE|LOGIN|LOGOUT} — filter by action type
  - ?module={model_name} — filter by model_name (case-insensitive contains), e.g. "User", "Customer", "Job"
  - ?date_from={YYYY-MM-DD} — filter entries on or after this date
  - ?date_to={YYYY-MM-DD} — filter entries on or before this date (inclusive, so add 1 day)
  - ?search={text} — search in object_repr (case-insensitive contains)
- Ordering: newest first (default, -timestamp)
- Return format: {count, next, previous, results} — same pagination pattern as user list

Add URL pattern to core/urls.py:
```
path("audit-log/", AuditLogListView.as_view(), name="audit_log_list"),
```
