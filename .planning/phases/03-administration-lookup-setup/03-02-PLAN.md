Build the user management page with the following components:

**page.tsx** — Page wrapper:
- "use client" directive
- Page title "User Management" with a "Create User" button (top right)
- Renders UserTable component
- Manages dialog state for create/edit/temp-password dialogs

**UserTable.tsx** — AG Grid data table:
- Use AG Grid Community (already installed) for the user list
- Columns: Username, Full Name (first + last), Email, Role (badge), Status (Active/Inactive badge), Last Login (formatted date), Actions (edit + deactivate/activate buttons)
- AG Grid column widths: Claude's discretion
- Server-side pagination: fetch /api/users/?page=N&page_size=20 using apiFetch
- Search bar above grid: debounced input that passes ?search= to API
- Active/Inactive filter toggle
- Action column: Edit button opens UserFormDialog in edit mode; Deactivate button calls POST /api/users/{id}/deactivate/ then refreshes list; Activate button calls POST /api/users/{id}/activate/
- Use apiFetch from @/lib/api for all API calls

**UserFormDialog.tsx** — Create/edit modal:
- Uses shadcn Dialog component
- React Hook Form + Zod validation
- Fields: Username (create only, disabled on edit), Email, First Name, Last Name, Role (dropdown select — fetch roles from GET /api/roles/), Phone (optional), Department (optional)
- On create submit: POST /api/users/ -> on success, close dialog, show TempPasswordDialog with the returned temp_password, refresh table
- On edit submit: PATCH /api/users/{id}/ -> on success, close dialog, refresh table
- Zod schema: username required (3-30 chars, alphanumeric+underscore), email required (valid email), first_name required, last_name required, role_id required

**TempPasswordDialog.tsx** — One-time temp password display:
- Modal showing the auto-generated temp password after user creation
- Clear instruction: "This password will only be shown once. Please copy and share it with the user."
- Copy-to-clipboard button
- "Done" button to close

Use the project design tokens: teal #1F7A8C for primary actions, amber #F89C1C for warnings, dark navy #2B3E50 for text. Use Tailwind classes throughout.
