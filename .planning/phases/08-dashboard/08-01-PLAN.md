Create backend/core/dashboard_views.py with two APIView classes: DashboardView and DashboardActivityView.

**DashboardView (GET /api/dashboard/):**

Permission class: IsAnyRole (imported from core.permissions).

KPI aggregation — compute all four metrics in a single GET handler:

1. active_jobs: Job.objects.filter(status__in=["PENDING", "IN_PROGRESS", "CUSTOMS", "DELIVERED"]).count()
   - Import Job from jobs.models

2. pending_approvals: ApprovalQueue.objects.filter(status="PENDING").count()
   - Import ApprovalQueue from approvals.models
   - If the requesting user's role is "Finance" (check request.user.profile.role.name), set pending_approvals = None in the response — Finance cannot act on approvals so the count is not surfaced

3. outstanding_invoice_total: Net balance across all non-PAID, non-CANCELLED invoices.
   Use Python-level aggregation:
   ```python
   from decimal import Decimal
   from django.db.models import Sum
   from accounts.models import Invoice, Payment

   invoices = Invoice.objects.exclude(status__in=["PAID", "CANCELLED"])
   invoiced = invoices.aggregate(total=Sum("amount"))["total"] or Decimal("0.00")
   paid = Payment.objects.filter(
       invoice__in=invoices
   ).aggregate(total=Sum("amount"))["total"] or Decimal("0.00")
   outstanding_invoice_total = str(invoiced - paid)   # string for JSON serialisation
   ```
   Note: Return as string (e.g. "12450.00") — the frontend will format it as "GHS 12,450.00".

4. new_customers_this_month: 
   ```python
   from django.utils import timezone
   from customers.models import Customer
   today = timezone.now()
   new_customers_this_month = Customer.objects.filter(
       created_at__year=today.year,
       created_at__month=today.month
   ).count()
   ```

Feed for first page — call the shared _build_feed helper (see below) with offset=0, limit=10.

Response shape:
```json
{
  "kpis": {
    "active_jobs": 12,
    "pending_approvals": 3,       // null for Finance role
    "outstanding_invoice_total": "12450.00",
    "new_customers_this_month": 5
  },
  "feed": {
    "count": 47,
    "next": "/api/dashboard/activity/?offset=10&limit=10",
    "previous": null,
    "results": [...]
  }
}
```

**DashboardActivityView (GET /api/dashboard/activity/):**

Permission class: IsAnyRole.
Reads ?offset and ?limit query params (default: offset=0, limit=10).
Calls _build_feed(request, offset, limit) and returns the paginated feed dict directly.

**Shared _build_feed(request, offset, limit) module-level function:**

Role-filtering logic — determine which AuditLog model_name values to include:
- Admin: all model_names (no filter)
- Finance: model_name__in=["Invoice", "Payment"]
- Operations: model_name__in=["Job", "Customer", "ApprovalQueue"]

Query:
```python
from core.models import AuditLog

role = request.user.profile.role.name
if role == "Admin":
    qs = AuditLog.objects.select_related("user").all()
elif role == "Finance":
    qs = AuditLog.objects.select_related("user").filter(model_name__in=["Invoice", "Payment"])
else:  # Operations
    qs = AuditLog.objects.select_related("user").filter(model_name__in=["Job", "Customer", "ApprovalQueue"])

total = qs.count()
entries = qs[offset:offset + limit]
```

ActivityEntry serialisation for each AuditLog entry:
```python
{
    "id": entry.pk,
    "source_type": entry.model_name,
    "actor_name": entry.user.get_full_name() or entry.user.username if entry.user else "System",
    "action": f"{entry.action} {entry.model_name} #{entry.object_id}",  // e.g. "CREATE Job #42"
    "timestamp": entry.timestamp.isoformat(),
    "link": _resolve_link(entry),  // see below
}
```

_resolve_link(entry) — returns a frontend path string based on model_name:
- "Job" → f"/jobs/{entry.object_id}"
- "Customer" → f"/customers/{entry.object_id}"
- "ApprovalQueue" → "/approvals"
- "Invoice" → "/accounts"
- "Payment" → "/accounts"
- default → None

Pagination envelope:
```python
next_url = f"/api/dashboard/activity/?offset={offset + limit}&limit={limit}" if (offset + limit) < total else None
prev_url = f"/api/dashboard/activity/?offset={max(0, offset - limit)}&limit={limit}" if offset > 0 else None

return {
    "count": total,
    "next": next_url,
    "previous": prev_url,
    "results": [serialise_entry(e) for e in entries],
}
```
