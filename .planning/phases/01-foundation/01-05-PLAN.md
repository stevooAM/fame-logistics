  <action>
    Create Django apps: `customers`, `jobs`. Add both to INSTALLED_APPS in settings.py.

    **customers/models.py**:
    - `Customer(TimeStampedModel)`:
      - company_name (CharField max 200)
      - customer_type (CharField choices: Company/Individual)
      - contact_person (CharField max 150, blank=True)
      - email (EmailField, blank=True)
      - phone (CharField max 50, blank=True) — accepts mixed local/international formats
      - tin (CharField max 50, blank=True, db_index=True) — supports C, P, G, Q, GHA, EUROPE prefixes
      - address (TextField, blank=True)
      - is_active (BooleanField, default=True) — soft delete flag
      - notes (TextField, blank=True)
      - Indexes: tin, company_name, customer_type, is_active
      - Meta: ordering = ['company_name']
      - __str__: company_name

    **jobs/models.py**:
    - `JobStatus` TextChoices: DRAFT, PENDING, IN_PROGRESS, CUSTOMS, DELIVERED, CLOSED, CANCELLED
    - `JobType` TextChoices: IMPORT, EXPORT, TRANSIT, LOCAL
    - `Job(TimeStampedModel)`:
      - job_number (CharField max 20, unique, db_index=True) — auto-generated e.g. FMS-2026-00001
      - customer (FK to Customer, on_delete=PROTECT)
      - job_type (CharField choices=JobType)
      - status (CharField choices=JobStatus, default=DRAFT)
      - origin (CharField max 200, blank=True)
      - destination (CharField max 200, blank=True)
      - cargo_description (TextField, blank=True)
      - bill_of_lading (CharField max 100, blank=True)
      - container_number (CharField max 50, blank=True)
      - weight_kg (DecimalField max_digits=12, decimal_places=3, null=True, blank=True)
      - volume_cbm (DecimalField max_digits=12, decimal_places=3, null=True, blank=True)
      - total_cost (DecimalField max_digits=15, decimal_places=2, null=True, blank=True)
      - notes (TextField, blank=True)
      - created_by (FK to User, on_delete=SET_NULL, null=True)
      - Indexes: job_number, status, job_type, customer, created_at

    - `JobAuditTrail(TimeStampedModel)`:
      - job (FK to Job, related_name='audit_trail')
      - user (FK to User, on_delete=SET_NULL, null=True)
      - action (CharField max 100) — e.g., "Status changed from DRAFT to PENDING"
      - old_value (CharField max 200, blank=True)
      - new_value (CharField max 200, blank=True)
      - Indexes: job, created_at

    - `JobDocument(TimeStampedModel)`:
      - job (FK to Job, related_name='documents')
      - document_type (FK to 'setup.DocumentType', null=True, on_delete=SET_NULL) — use string reference since setup app is created in 01-07
      - file_name (CharField max 255)
      - file_url (URLField) — cloud storage URL
      - file_size (PositiveIntegerField, null=True)
      - uploaded_by (FK to User, on_delete=SET_NULL, null=True)

    Register all models in their respective `admin.py` files with basic ModelAdmin (list_display with key fields).

    After defining all models, generate and apply migrations:
    ```bash
    docker compose exec backend python manage.py makemigrations core customers jobs
    docker compose exec backend python manage.py migrate
    ```

    Note: JobDocument has a string FK reference to 'setup.DocumentType' which will resolve once 01-07 creates the setup app. Use the string form `ForeignKey('setup.DocumentType', ...)` so Django defers resolution.
