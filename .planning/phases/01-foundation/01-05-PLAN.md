  <action>
    Create Django apps: `customers`, `jobs`, `approvals`, `accounts`, `setup`. Add all to INSTALLED_APPS in settings.py.

    **core/models.py** — Shared base and auth extension:
    - `TimeStampedModel` (abstract): created_at (auto_now_add), updated_at (auto_now) — all models inherit this
    - `AuditLog`: user (FK to User, nullable for system actions), action (CharField: CREATE/UPDATE/DELETE), model_name, object_id, object_repr, changes (JSONField), ip_address (GenericIPAddressField, nullable), timestamp (auto_now_add)
    - `Role`: name (CharField, unique, choices: Admin/Manager/Operations/Finance/Viewer), description
    - `UserProfile`: user (OneToOneField to auth.User), role (FK to Role), is_active (BooleanField, default True), phone, department

    **customers/models.py**:
    - `Customer(TimeStampedModel)`:
      - company_name (CharField max 200)
      - customer_type (CharField choices: Company/Individual)
      - contact_person (CharField max 150, blank=True)
      - email (EmailField, blank=True)
      - phone (CharField max 50, blank=True) — accepts mixed local/international formats
      - tin (CharField max 50, blank=True, db_index=True) — supports C, P, G, Q, GHA, EUROPE prefixes
      - address (TextField, blank=True)
      - is_active (BooleanField, default=True) — soft delete flag
      - notes (TextField, blank=True)
      - Indexes: tin, company_name, customer_type, is_active
      - Meta: ordering = ['company_name']
      - __str__: company_name

    **jobs/models.py**:
    - `JobStatus` TextChoices: DRAFT, PENDING, IN_PROGRESS, CUSTOMS, DELIVERED, CLOSED, CANCELLED
    - `JobType` TextChoices: IMPORT, EXPORT, TRANSIT, LOCAL
    - `Job(TimeStampedModel)`:
      - job_number (CharField max 20, unique, db_index=True) — auto-generated e.g. FMS-2026-00001
      - customer (FK to Customer, on_delete=PROTECT)
      - job_type (CharField choices=JobType)
      - status (CharField choices=JobStatus, default=DRAFT)
      - origin (CharField max 200, blank=True)
      - destination (CharField max 200, blank=True)
      - cargo_description (TextField, blank=True)
      - bill_of_lading (CharField max 100, blank=True)
      - container_number (CharField max 50, blank=True)
      - weight_kg (DecimalField max_digits=12, decimal_places=3, null=True, blank=True)
      - volume_cbm (DecimalField max_digits=12, decimal_places=3, null=True, blank=True)
      - total_cost (DecimalField max_digits=15, decimal_places=2, null=True, blank=True)
      - notes (TextField, blank=True)
      - created_by (FK to User, on_delete=SET_NULL, null=True)
      - Indexes: job_number, status, job_type, customer, created_at

    - `JobAuditTrail(TimeStampedModel)`:
      - job (FK to Job, related_name='audit_trail')
      - user (FK to User, on_delete=SET_NULL, null=True)
      - action (CharField max 100) — e.g., "Status changed from DRAFT to PENDING"
      - old_value (CharField max 200, blank=True)
      - new_value (CharField max 200, blank=True)
      - Indexes: job, created_at

    - `JobDocument(TimeStampedModel)`:
      - job (FK to Job, related_name='documents')
      - document_type (FK to setup.DocumentType, null=True, on_delete=SET_NULL)
      - file_name (CharField max 255)
      - file_url (URLField) — cloud storage URL
      - file_size (PositiveIntegerField, null=True)
      - uploaded_by (FK to User, on_delete=SET_NULL, null=True)

    **approvals/models.py**:
    - `ApprovalQueue(TimeStampedModel)`:
      - job (FK to Job, related_name='approval_requests')
      - submitted_by (FK to User, related_name='submitted_approvals', on_delete=SET_NULL, null=True)
      - status (CharField choices: PENDING/APPROVED/REJECTED, default=PENDING)
      - reviewed_by (FK to User, related_name='reviewed_approvals', on_delete=SET_NULL, null=True, blank=True)
      - reviewed_at (DateTimeField, null=True, blank=True)
      - rejection_reason (TextField, blank=True)
      - Indexes: status, submitted_by, reviewed_by

    - `ApprovalHistory(TimeStampedModel)`:
      - approval (FK to ApprovalQueue, related_name='history')
      - action (CharField: SUBMITTED/APPROVED/REJECTED)
      - actor (FK to User, on_delete=SET_NULL, null=True)
      - comment (TextField, blank=True)

    **accounts/models.py**:
    - `Invoice(TimeStampedModel)`:
      - invoice_number (CharField max 20, unique, db_index=True) — auto-generated
      - job (FK to Job, related_name='invoices')
      - customer (FK to Customer) — denormalized for query convenience
      - amount (DecimalField max_digits=15, decimal_places=2)
      - currency (FK to setup.Currency, null=True, on_delete=SET_NULL)
      - status (CharField choices: DRAFT/SENT/PAID/PARTIAL/OVERDUE/CANCELLED, default=DRAFT)
      - issue_date (DateField)
      - due_date (DateField)
      - notes (TextField, blank=True)
      - created_by (FK to User, on_delete=SET_NULL, null=True)
      - Indexes: invoice_number, status, customer, issue_date

    - `Payment(TimeStampedModel)`:
      - invoice (FK to Invoice, related_name='payments')
      - amount (DecimalField max_digits=15, decimal_places=2)
      - payment_date (DateField)
      - payment_method (CharField max 50, blank=True)
      - reference (CharField max 100, blank=True) — cheque/transfer ref
      - notes (TextField, blank=True)
      - recorded_by (FK to User, on_delete=SET_NULL, null=True)

    **setup/models.py** — Lookup tables:
    - `Port(TimeStampedModel)`: name (CharField max 200, unique), code (CharField max 20, unique), country (CharField max 100), is_active (BooleanField, default=True)
    - `CargoType(TimeStampedModel)`: name (CharField max 100, unique), description (TextField, blank=True), is_active (BooleanField, default=True)
    - `Currency(TimeStampedModel)`: code (CharField max 3, unique), name (CharField max 100), symbol (CharField max 5), exchange_rate (DecimalField max_digits=12, decimal_places=6, default=1), is_default (BooleanField, default=False), is_active (BooleanField, default=True)
    - `DocumentType(TimeStampedModel)`: name (CharField max 100, unique), description (TextField, blank=True), is_active (BooleanField, default=True)
    - `CompanyProfile(TimeStampedModel)`: name (CharField max 200), logo_url (URLField, blank=True), email (EmailField, blank=True), phone (CharField max 50, blank=True), address (TextField, blank=True), website (URLField, blank=True), tin (CharField max 50, blank=True)
      - Use a singleton pattern: override save() to ensure only one record exists

    Register all models in their respective `admin.py` files with basic ModelAdmin (list_display with key fields).
