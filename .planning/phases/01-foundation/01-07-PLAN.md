  <action>
    Create Django apps: `approvals`, `accounts`, `setup`. Add all three to INSTALLED_APPS in settings.py.

    **approvals/models.py**:
    - `ApprovalQueue(TimeStampedModel)`:
      - job (FK to Job, related_name='approval_requests')
      - submitted_by (FK to User, related_name='submitted_approvals', on_delete=SET_NULL, null=True)
      - status (CharField choices: PENDING/APPROVED/REJECTED, default=PENDING)
      - reviewed_by (FK to User, related_name='reviewed_approvals', on_delete=SET_NULL, null=True, blank=True)
      - reviewed_at (DateTimeField, null=True, blank=True)
      - rejection_reason (TextField, blank=True)
      - Indexes: status, submitted_by, reviewed_by

    - `ApprovalHistory(TimeStampedModel)`:
      - approval (FK to ApprovalQueue, related_name='history')
      - action (CharField: SUBMITTED/APPROVED/REJECTED)
      - actor (FK to User, on_delete=SET_NULL, null=True)
      - comment (TextField, blank=True)

    **accounts/models.py**:
    - `Invoice(TimeStampedModel)`:
      - invoice_number (CharField max 20, unique, db_index=True) — auto-generated
      - job (FK to Job, related_name='invoices')
      - customer (FK to Customer) — denormalized for query convenience
      - amount (DecimalField max_digits=15, decimal_places=2)
      - currency (FK to setup.Currency, null=True, on_delete=SET_NULL)
      - status (CharField choices: DRAFT/SENT/PAID/PARTIAL/OVERDUE/CANCELLED, default=DRAFT)
      - issue_date (DateField)
      - due_date (DateField)
      - notes (TextField, blank=True)
      - created_by (FK to User, on_delete=SET_NULL, null=True)
      - Indexes: invoice_number, status, customer, issue_date

    - `Payment(TimeStampedModel)`:
      - invoice (FK to Invoice, related_name='payments')
      - amount (DecimalField max_digits=15, decimal_places=2)
      - payment_date (DateField)
      - payment_method (CharField max 50, blank=True)
      - reference (CharField max 100, blank=True) — cheque/transfer ref
      - notes (TextField, blank=True)
      - recorded_by (FK to User, on_delete=SET_NULL, null=True)

    **setup/models.py** — Lookup tables:
    - `Port(TimeStampedModel)`: name (CharField max 200, unique), code (CharField max 20, unique), country (CharField max 100), is_active (BooleanField, default=True)
    - `CargoType(TimeStampedModel)`: name (CharField max 100, unique), description (TextField, blank=True), is_active (BooleanField, default=True)
    - `Currency(TimeStampedModel)`: code (CharField max 3, unique), name (CharField max 100), symbol (CharField max 5), exchange_rate (DecimalField max_digits=12, decimal_places=6, default=1), is_default (BooleanField, default=False), is_active (BooleanField, default=True)
    - `DocumentType(TimeStampedModel)`: name (CharField max 100, unique), description (TextField, blank=True), is_active (BooleanField, default=True)
    - `CompanyProfile(TimeStampedModel)`: name (CharField max 200), logo_url (URLField, blank=True), email (EmailField, blank=True), phone (CharField max 50, blank=True), address (TextField, blank=True), website (URLField, blank=True), tin (CharField max 50, blank=True)
      - Use a singleton pattern: override save() to ensure only one record exists

    Register all models in their respective `admin.py` files with basic ModelAdmin (list_display with key fields).

    After defining all models, generate and apply migrations:
    ```bash
    docker compose exec backend python manage.py makemigrations approvals accounts setup
    docker compose exec backend python manage.py migrate
    ```

    Note: The setup.DocumentType model resolves the string FK reference in jobs.JobDocument (created in 01-05). After this migration, the full FK graph is complete.
